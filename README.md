## Description

* A minimalist todo list utility with tiny size.

## Getting Started

### Dependencies

* chalk

### Installing

* npm install

### Executing program

* source ./alias
* todo init tasklist
* todo add task1
* todo done 0       (use id 0 instead of "task1")
* todo undone 0
* todo toggle 0
* todo remove 0
* todo list
